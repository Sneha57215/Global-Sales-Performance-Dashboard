# Global-Sales-Performance-Dashboard
This project showcases an end-to-end Power BI solution built using a real-world sales dataset

Key Features:

Comprehensive dashboard covering Revenue, Profit, Quantity, and Order trends

DAX Measures for Profit, Margin %, YoY Growth, and YTD metrics

Channel-wise, Region-wise, and Product Category analysis

Interactive visuals with drill-through, tooltips, and dynamic filters

Data cleaning and transformation performed using Power Query

Files Included:

.pbix dashboard file

Dataset (CSV/XLSX)

Objective:

Provide actionable insights into overall business performance and sales opportunities.
